2AI 적용 가능 · 조건부 · 부적합 영역 가이드

설문 배포 시 아래 가이드를 함께 안내하여, 현실적인 니즈가 취합되도록 합니다.

◎ 즉시 적용 가능
  • 회의록·보고서·이메일 초안 작성, 문서 요약·번역
  • ERP 데이터 조회·요약 리포트 (읽기 전용 연계)
  • 매출·재고 등 정기 보고서 자동 생성
  • 사내 규정·매뉴얼 질의응답 챗봇
  • 엑셀 데이터 정리·시각화 보조
  • 단순 반복 취합 업무(주간보고 통합 등)
△ 조건부 가능 (통제 필요)
  • 수요예측·이상치 탐지 등 분석 — 데이터 품질·검증 절차 전제
  • 거래처·계약 문서 검토 보조 — 최종 판단은 담당자
  • 개인정보 포함 업무 — 마스킹·내부 처리 전제
  • ERP 데이터 "입력" 자동화 — ERP 표준 인터페이스(PDW 등) 경유 시에만
✕ 부적합 / 불가
  • ERP 원장 직접 수정·삭제 (회계·재고·생산 데이터)
  • 결산·세무신고 수치의 무검증 자동 확정
  • 인사평가·급여 결정 등 사람에 대한 최종 판단
  • 법적 효력 문서의 무검토 발송
  • 사내 기밀을 통제되지 않은 외부 AI에 입력
전사 공통 원칙 AI는 초안 작성·조회·요약·분석 보조까지 수행하고, 최종 확인과 책임은 반드시 담당자에게 있습니다. ERP 원장 데이터는 AI가 직접 수정하지 않으며(읽기 전용), 사내 데이터는 회사가 승인한 채널에서만 사용합니다.

5취합 후 과제 평가 기준

회수된 니즈는 기대효과(절감시간×인원) × 구현 난이도(데이터 위치·보안 민감도) 2축으로 평가하여 우선순위를 정합니다.

가로축: 구현 난이도 (왼쪽 = 쉬움) / 세로축: 기대효과 (위 = 큼)
① 우선 추진 (효과 大 · 난이도 低)정기 리포트 자동화, 문서 초안, 규정 챗봇 등 → PoC 대상
② 중기 추진 (효과 大 · 난이도 高)ERP 연계 분석·예측 등 → 2~3단계 과제화
③ 여유 시 추진 (효과 小 · 난이도 低)개인 생산성 도구 안내·교육으로 대체
④ 보류 (효과 小 · 난이도 高)또는 부적합 영역(원장 수정 등) → 제외 사유 회신
평가 시 가점/감점 가점: 여러 부서 공통 수요, 기존 데이터(ERP)만으로 구현 가능, 측정 가능한 효과. 감점: 개인정보·기밀 민감도 높음, 외부 데이터 신규 확보 필요, 법적 검토 필요.

2주차

4주차
