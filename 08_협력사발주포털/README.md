## 데모 실행

`02_데모/` 의 두 파일을 **같은 브라우저**에서 두 탭으로 열면 실시간 연동됩니다.
모바일에서 사진 등록·검수요청 → 대시보드 반영, 대시보드에서 검수 판정(합/부)·메시지 → 모바일 반영.
