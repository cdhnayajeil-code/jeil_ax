🗄️ ERP 중간 DB 샘플 데이터 erp_ro · 읽기전용

UNIERP(사외 IDC MSSQL) 원천을 야간 배치로 적재한 PostgreSQL 중간 DB(erp_ro) 스키마를 테이블 형태로 샘플링한 자료입니다. 대시보드·챗봇은 이 중간 DB만 조회합니다(운영 ERP 직접 연결 없음). 발주 10건 · 매입(구매) 10건 + 연관 마스터/이력 테이블.

📅 데이터 기준: · ETL 배치
구성 방식 — 설계 문서 07_ERP_챗봇_중간DB_연계설계 기준. 중간 DB는 원본 전체 복제가 아니라 대시보드·챗봇이 바로 쓰는 집계·요약 + 키 정합 형태로 적재되며, 모든 테이블에 공통 컬럼 synced_at · src_updated · batch_id 를 둡니다.
