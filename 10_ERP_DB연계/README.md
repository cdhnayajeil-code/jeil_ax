## 미러 동기화 규칙 (★중요)

이 폴더의 문서는 사내 OneDrive **`최동혁/ERP_DB/정책관리/ERP_DB연계/`** 폴더로 미러링한다(경영·정책 보고용 사본). 동기화는 전담 에이전트 `erp-db-link-manager`가 수행하며 규칙은 다음과 같다.

1. **원본(단일 출처)은 본 폴더**. 미러 쪽을 직접 수정하지 않는다(수정 필요 시 원본 수정 → 재동기화).
2. `02_진행상태`는 미러에서 **버전 없는 단일 파일**(`ERP_DB연계_진행상태.md/.html`)로 항상 최신 유지.
3. `00_현재상태`·`01_연계기획`은 **의미 있는 개정 시** 버전 사본(`ERP_DB연계_<문서명>_v<버전>_<YYYYMMDD>.md/.html`)을 추가하고 구버전은 보존(정책관리 폴더 관례).
4. 동기화 시 `정책관리/README.md`의 프로젝트 목록 표(최신 버전)를 갱신한다.
